🔄 تحديث أيقونات PWA

دليل شامل لحل مشكلة عدم تحديث الأيقونات في التطبيقات التقدمية

🔍 تشخيص المشكلة

عندما تقوم بتحديث أيقونات PWA ولا تظهر التغييرات، فإن السبب عادة ما يكون:

  • التخزين المؤقت للمتصفح: المتصفح يحفظ الأيقونات القديمة
  • Service Worker Cache: الـ Service Worker يخزن الأصول القديمة
  • عدم تطابق المسارات: مسارات الأيقونات في manifest.json لا تطابق الملفات الفعلية
  • عدم تحديث رقم الإصدار: لم يتم تغيير رقم إصدار الكاش

🛠️ الحلول التقنية

Google Chrome / Microsoft Edge

1
فتح أدوات المطور
اضغط F12 أو Ctrl+Shift+I (Cmd+Option+I على Mac)
2
الانتقال إلى تبويب Application
ابحث عن تبويب "Application" في أدوات المطور
3
مسح Storage
في الشريط الجانبي، اختر "Storage" ثم "Clear storage"
4
مسح جميع البيانات
تأكد من تحديد جميع الخيارات ثم اضغط "Clear site data"
5
إعادة تحميل الصفحة
اضغط Ctrl+Shift+R للتحديث مع تجاهل الكاش

Mozilla Firefox

1
فتح أدوات المطور
اضغط F12 أو Ctrl+Shift+I
2
الانتقال إلى Storage
اختر تبويب "Storage" من أدوات المطور
3
مسح Cache Storage
انقر بالزر الأيمن على "Cache Storage" واختر "Delete All"
4
مسح Service Workers
في "Service Workers"، انقر "Unregister" لكل service worker

Safari

1
تفعيل Developer Menu
Safari → Preferences → Advanced → Show Develop menu
2
مسح الكاش
Develop → Empty Caches
3
إعادة التحميل
اضغط Cmd+Shift+R للتحديث مع تجاهل الكاش

الهاتف المحمول

1
Android Chrome
Settings → Privacy and security → Clear browsing data → Advanced → تحديد "Cached images and files" و "Site settings"
2
iOS Safari
Settings → Safari → Clear History and Website Data
3
إعادة تثبيت PWA
احذف التطبيق من الشاشة الرئيسية وأعد تثبيته

⚡ الحل السريع (للمطورين)

يمكنك استخدام الكود التالي في وحدة التحكم (Console) لمسح الكاش برمجياً:

// مسح جميع الكاش await caches.keys().then(names => { return Promise.all(names.map(name => caches.delete(name))); }); // إلغاء تسجيل Service Workers await navigator.serviceWorker.getRegistrations().then(registrations => { return Promise.all(registrations.map(registration => registration.unregister())); }); // إعادة تحميل الصفحة window.location.reload(true);
⚠️ تنبيه: هذا الكود سيمسح جميع البيانات المحفوظة محلياً للموقع

🔧 للمطورين: منع المشكلة مستقبلاً

1. تحديث رقم إصدار الكاش

const CACHE_NAME = 'my-app-v1.0.1'; // زيادة الرقم مع كل تحديث

2. إضافة Cache Busting

"start_url": "/?v=1.0.1", "icons": [ { "src": "/icons/icon-192x192.png?v=1.0.1", "sizes": "192x192", "type": "image/png" } ]

3. تحديث Service Worker تلقائياً

// في Service Worker self.addEventListener('install', event => { self.skipWaiting(); // تفعيل فوري للإصدار الجديد }); self.addEventListener('activate', event => { event.waitUntil( caches.keys().then(cacheNames => { return Promise.all( cacheNames.map(cacheName => { if (cacheName !== CURRENT_CACHE) { return caches.delete(cacheName); } }) ); }) ); });
